NestJS에서 사용하는 기본 구조를 바탕으로 GraphQL 및 TypeORM과 함께 사용할 때 빠르고 직관적이면서도, 유연하고 확장성있는 코드를 개발하기 위한 구조 예시입니다.

# To start

## 0. 환경변수 세팅

```
Github -> Settings -> Secrets&Variables 들어가서 복사 후 파일 생성하여 사용

```

## 1. 도커 컨테이너 구동

```

yarn docker:local

```

## 2. 서버 시작

```

yarn start:local
```
